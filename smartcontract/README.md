This is a smart contract developed by using Hardhat.

To init the project, run command:

`pnpm install`

To run the test cases, run command:

`pnpm hardhat test`

`pnpm hardhat test solidity`

`pnpm hardhat test nodejs`

To run local network and deploy contract on it, run command:

`pnpm hardhat node`

`pnpm hardhat run --network localhost scripts/deploy.ts`
